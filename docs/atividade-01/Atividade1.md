# Atividade 01 — Análise de Qualidade do Sistema Local Eats

| Problema identificado | Atributo de qualidade afetado | Justificativa técnica |
|---|---|---|
| Lentidão em horários de pico | Desempenho (Performance Efficiency) | Tempo de resposta elevado prejudica a experiência do usuário e indica que o sistema não está preparado para muitos acessos simultâneos. |
| Telas confusas e pouco intuitivas | Usabilidade | Interfaces pouco claras dificultam a navegação e tornam o sistema difícil de usar. |
| Buscas retornam resultados incorretos | Adequação Funcional | A funcionalidade de busca não entrega os resultados esperados pelo usuário. |
| Aplicativo apresenta falhas em alguns smartphones | Compatibilidade | O sistema não funciona corretamente em diferentes dispositivos ou versões de sistema operacional. |
| Usuários têm dificuldade para concluir ações simples | Usabilidade | Fluxos mal projetados tornam tarefas simples mais difíceis de executar. |
| Avaliações desaparecem após atualizar a página | Confiabilidade | A perda de dados indica possível falha no armazenamento ou recuperação das informações. |
| Inconsistência entre versão web e mobile | Compatibilidade | Diferenças entre plataformas geram comportamento inconsistente e confusão para o usuário. |
