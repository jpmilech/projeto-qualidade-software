## 🎯 Objetivo do Repositório
Este repositório reúne as atividades desenvolvidas na disciplina de **Qualidade de Software** do curso de **Análise e Desenvolvimento de Sistemas / Sistemas para Internet** do **Centro Universitário Senac-RS**.

As atividades são realizadas utilizando a metodologia **PBL (Problem-Based Learning — Aprendizagem Baseada em Problemas)**, com foco na análise de situações reais da área de desenvolvimento de software.

## 📚 Organização do Projeto

- `docs/` → Atividades realizadas ao longo das aulas
- `artefatos/` → Diagramas, relatórios e evidências
- `src/` → Código-fonte (quando houver implementação)
- `referencias/` → Materiais de apoio e bibliografia

## 📦 Entregas

As atividades estão organizadas dentro da pasta `docs/`:

- **atividade-01** → Análise de qualidade do sistema *Local Eats*, identificando problemas do produto e relacionando-os com atributos de qualidade da norma **ISO/IEC 25010**.

## 🧩 Metodologia

O projeto segue a metodologia **PBL (Aprendizagem Baseada em Problemas)**, na qual os estudantes analisam e resolvem problemas práticos relacionados à qualidade de software.
